# ASHA THE HOPE

Project is split into separate apps:

- `frontend/` - React client
- `backend/` - Express + MongoDB API

## Run Frontend

```bash
cd frontend
npm install
npm start
```

## Run Backend

```bash
cd backend
npm install
npm run dev
```

## Environment Files

- Frontend: `frontend/.env`
- Backend: `backend/.env`

Use `.env.example` in each folder as template.
